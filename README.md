在Terminal中 cd 檔案目錄，先於目標位置在命令行執行 python manage.py migrate ，
接著執行python manage.py makemigrations，
再執行 python manage.py migrate，
然後在命令行中執行 python manage.py runserver</br></br> 在瀏覽器中輸入顯示的端口號，例如 127.0.0.1:8000</br>
